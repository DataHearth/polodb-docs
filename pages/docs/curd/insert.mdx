# Insert

Invoke the `insert` method to insert an document.
If the `_id` is not provided, PoloDB will mutate the document to
add an id.

```rust
let mut db = Database::open_file("/tmp/test-collection").unwrap();
let mut collection = db.collection("test").unwrap();
collection.insert(doc! {
    "_id": 0,
    "name": "Vincent Chan",
    "score": 99.99,
}.as_mut());
```
